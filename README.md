# 🌾 AI-Powered Smart Crop Recommendation System

This project recommends the most suitable crop based on soil and climate conditions using Machine Learning.

## 🚀 Technologies Used
- Python
- Scikit-learn
- Pandas
- Streamlit

## 📊 Models Used
- Logistic Regression
- Random Forest

## 🔧 How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Run model:
   python model.py

3. Run app:
   streamlit run app.py
